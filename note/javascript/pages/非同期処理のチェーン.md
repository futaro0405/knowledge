複数の非同期処理をcallback関数を使って連続的につなげて処理するための方法
callback関数を使って非同期処理のチェーンを行う

```js
function sleep(callback, val) {
	setTimeout(function() {
		console.log(val++);
		callback(val);
	}, 1000);
}

sleep(function(val) {
	sleep(function(val) {
		sleep(function(val) {
			sleep(function(val) {
			}, val);
		}, val);
	}, val);
}, 0);
```

```console
0
1
2
3
```
