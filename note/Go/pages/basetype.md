integer型

```go
var i int = 100
```

int型にも種類がある
最大値、最小値に違いがある。
PCの環境依存になる。

`int0`: -128 to 127
`int16`: -32768 to 32767
`int32`: -2147483648 to 2147483647
`int64`: -9223372036854775808 to 9223372036854775807

明示的に指定することができるが型が異なるものと判定されるため注意が必要

```go
var i int = 100
var i2 int64 = 120

fmt.Println(i + i2)
// error
```

`fmt.Printf()`はフォーマットを指定してprintを実行できる
```
fmt.Printf("%T\n", i2)
```
