# ユニオン型とリテラル型
特定の変数や引数に複数の型が入ることが想定される場合、**ユニオン型**を使用する。
文字列型と数値型が想定される場合は、`string | number`のように記述する。

```index.ts
const outputNumOrStr = (val: string | number) => {
  console.log(val);
};
```

また、上記を利用することで、特定の値だけを代入可能にすることができる。
そのようなかたを**リテラル型**という。
例えば、10と20のみ想定される場合は、`10 | 20`と記述する。

```index.ts
let num1: 10 | 20 = 10;
```
